# Analysis Scripts

This directory contains analytical tools and scripts used throughout the repetition causality investigation.

## Overview

These scripts provide analytical capabilities, data processing, and result interpretation tools that supported the systematic investigation across all experimental phases.

## Script Categories

### Data Analysis
- **`attention_entropy_analysis.py`**: Entropy-based attention pattern analysis
- **`analysis_utils.py`**: Core analytical utilities and helper functions

### Results Processing
- Scripts for processing experimental outputs and generating insights
- Statistical analysis tools for evaluating intervention effectiveness
- Comparative analysis between different experimental approaches

These analysis scripts were instrumental in:
1. **Pattern Discovery**: Identifying attention patterns that led to breakthrough insights
2. **Results Validation**: Confirming experimental outcomes across all phases
3. **Comparative Analysis**: Evaluating relative effectiveness of different approaches
4. **Statistical Processing**: Rigorous analysis of experimental data
