# Source Code Directory

This directory contains the core source code infrastructure for the repetition causality research project.

## Directory Structure

### Core Infrastructure
- **`config/`**: Configuration files and parameters for experiments
- **`modules/`**: Core utility modules and shared libraries
- **`plots/`**: Plotting and visualization utilities
- **`experiments/`**: Core experiment infrastructure

### Experiment Categories
- **`early_interventions/`**: Direct intervention experiments (Phase 1-3)
  - Activation patching approaches
  - Causal attention interventions
  - Success rate: ~0% (critical negative results)

- **`analysis_scripts/`**: Analytical tools and data processing
  - Attention entropy analysis
  - Results validation utilities
  - Statistical processing frameworks

- **`comprehensive_tests/`**: Validation and testing framework
  - Comprehensive technique validation
  - Baseline comparison protocols
  - Deployment readiness assessment
