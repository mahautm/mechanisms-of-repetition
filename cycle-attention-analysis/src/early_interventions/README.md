# Early Intervention Experiments

This directory contains experiments from the early phases of repetition causality research, focusing on direct intervention approaches.

## Overview

These experiments represent systematic attempts to induce repetition through direct model interventions before the breakthrough discovery of semantic repetition patterns. While unsuccessful in achieving the target objective, they provided critical negative results that guided research toward the true causal mechanisms.

## Experiment Categories

### Activation Patching
- **`activation_patching_single_token.py`**: Tests single-token activation interventions
- **`activation_patching_multi_token.py`**: Multi-token activation manipulation experiments
- **`activation_patching_validate.py`**: Validation framework for activation-based approaches

### Causal Interventions
- **`causal_attention_intervention.py`**: Direct attention mechanism interventions
- Other causal intervention experiments exploring gradient-based and embedding manipulation approaches

## Key Findings

- **Success Rate**: ~0% for reliable repetition induction
- **Critical Insight**: Direct interventions cannot reliably trigger repetitive generation patterns
- **Research Value**: These negative results were essential for redirecting focus toward semantic pattern causality
- **Methodology**: Established rigorous intervention testing framework used in later successful experiments

## Scientific Impact

While these experiments did not achieve repetition induction, they:
1. Ruled out direct intervention approaches systematically
2. Established baseline testing methodology
3. Demonstrated need for alternative causal mechanisms
4. Provided comparative context for later breakthrough results (993.3x effectiveness)

## Connection to Breakthrough

These early failures directly motivated the investigation into semantic repetition patterns that ultimately achieved breakthrough success in phases 4-6.
